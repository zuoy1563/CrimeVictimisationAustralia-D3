FAQ

Why I cannot see the graphs?

Data is processed using json files, so it is necessary to have a server to load json files. Simply put, you can run the application using Brackets or other IDEs such as Webstorm, they can setup a local server for you while running the web pages.

How can I see the exact value at each point?

The exact value of each point can be shown by hovering on the point. A tooltip contains all the available information will pop up.

How can I see the victimisation trend for Australia after selecting a state on the map?

Simply click the "Reset to Australia" button, the trend will be automatically refreshed.
